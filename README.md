# Video Game Market Analysis Dashboard

Interactive dashboard analyzing video game sales, quality metrics, and market segmentation using K-Means clustering.

## Features
- Market segmentation with K-Means clustering
- Regional sales analysis
- Review score vs sales correlation
- Genre evolution over time
- Interactive filters and brushing & linking
